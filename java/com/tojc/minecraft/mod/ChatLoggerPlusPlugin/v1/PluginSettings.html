PluginSettings¶

public interface PluginSettings¶

プラグインの設定や登録を行うためのインターフェイスです。

Author:Jaken

Methods¶

registerPermissionAddAfterMessage¶

public void registerPermissionAddAfterMessage()¶

チャットメッセージを追加するパーミッションを要求します。 スクリプトで ChatMessage.addAfterMessage(String) を使用する場合は宣言が必要です。

registerPermissionMessageModification¶

public void registerPermissionMessageModification()¶

チャットメッセージを加工するパーミッションを要求します。 スクリプトで ChatMessage.setMessage(String) を使用する場合は宣言が必要です。

registerPermissionReadStack¶

public void registerPermissionReadStack(String keyname)¶

スタックを読み込むパーミッションを要求します。（実質、既に存在するスタックなら読み書きできます） スクリプトで ChatMessage.readStack(String) を使用する場合は宣言が必要です。

パラメタ:
  • keyname – アクセス対象のスタックから読み取るキー文字列

registerPermissionWriteStack¶

public void registerPermissionWriteStack(String keyname)¶

スタックに書き込むパーミッションを要求します。（実質新規作成の意味合いです） スクリプトで ChatMessage.writeStack(String,Object) を使用する場合は宣言が必要です。

パラメタ:
  • keyname – アクセス対象のスタックに登録するキー文字列
